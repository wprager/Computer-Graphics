Homework 3

This project was the culmination of all the previous assignments. We constructed a hedge maze scene where the program was able to read in .obj object files, .raw triangle files, and textures. We were also able to adjust color and shininess, have the objects correctly interact with lighting (directional and point lights), and even put in some basic animation. We also implemented a movement system that uses the mouse and keyboard so that it behaves similar to a first-person video game.
